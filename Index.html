Garbage Object Detection

This project focuses on object detection using the TACO (Trash Annotations for Clean Objects) dataset. By applying deep learning techniques and utilizing models like YOLOv8, it identifies and classifies various types of trash in images. The goal is to automate waste classification and improve recycling efforts, contributing to environmental sustainability.

Hate Speech Detection

This project uses a Bidirectional LSTM model to detect and classify hate speech in text. By leveraging deep learning and natural language processing techniques, it identifies harmful content in tweets. The model is trained and evaluated for optimal performance in accurately distinguishing between hateful and non-hateful speech.
